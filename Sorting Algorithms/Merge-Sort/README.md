# Merge Sort

Merge sort is a very quick way to sort array/lists. It uses the Conquer and Divide strategy. 

## Complexity Table

| Best            | Average             | Worst               | Memory    |
| :-------------: | :-----------------: | :-----------------: | :-------: |
| n&nbsp;log(n)   | n&nbsp;log(n)       | n&nbsp;log(n)       | n         |
